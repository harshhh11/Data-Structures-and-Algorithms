# Time Complexity

## Linked List 

| Operation | Time Complexity |
| ----------- | ----------- |
| Insert at begining | O(1) |
| Insert at the end | O(N) |
| Insert at i<sup>th</sup> Position | O(N) |
| Access | O(N) |


## Array 
| Operation | Time Complexity |
| ----------- | ----------- |
| Insert at Beginning | O(N) |
| Insert at End | O(1) if array is not full |
| Insert at End | O(N) if array is full |
| Insert at i<sup>th</sup> Position | O(N) |
| Access | O(1) |
